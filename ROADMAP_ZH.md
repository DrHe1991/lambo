# BitLink 产品路线图

> 内容预测市场，让优质内容得到回报

---

## 核心机制

1. **动态点赞定价**：赞越少越贵，赞越多越便宜
2. **早期点赞者分成**：50% 作者 + 40% 早期点赞者 + 10% 平台
3. **1小时锁定期**：点赞后锁定1小时，期间可撤回（退90%），到期后自动结算

## 经济机制详情

### 帖子点赞定价公式

```
cost = max(5, 100 / sqrt(1 + likes))

| 点赞数 | 成本 |
|--------|------|
|   0    | 100  |
|   5    |  41  |
|  20    |  22  |
| 100    |  10  |
| 500+   |   5  |
```

### 评论点赞定价公式

```
cost = max(2, 20 / sqrt(1 + likes))

| 点赞数 | 成本 |
|--------|------|
|   0    |  20  |
|   5    |   8  |
|  20    |   4  |
|  50+   |   2  |
```

### 收益分配

```
点赞收益 = 50% 作者 + 40% 早期点赞者 + 10% 平台
评论收费 = 100% 内容创作者
```
